Injections

SQL, No-SQL, Code, OS, XML, XPath etc.

What is the problem?

Executing untrusted or manipulated commands, requests or queries by your web application

Risks

  • Data loss, corruption, theft
  • Unauthorized access
  • Takeover of system by an attacker

Cross-Site Scripting (XSS)

What is the problem?

An attacker can inject untrusted snippets of JavaScript into your application

Types

  • Reflected XSS - malicious link
  • Stored XSS - Injected into application server

Risks

  • Account theft
  • Redirect to other malicious site
  • Modification of user data/content

What is the problem?

  • Attacker may recreate any traffic he wants

Broken Authentication and Session Management

What is the problem?

Attacker can capture or bypass authentication methods of web applications

- kradziez id sesji z linka - publiczny komputer - haslo nie shashowane

Types:

  • Session Management
  • Password guessing (brute-force attack)

Risks

  • Attacker may authorize as you to application
  • Attacker may steal your password if not properly encrypted

REDoS

What is the problem?

When untrusted data input is executed on a regex pattern, it may exploit vulnerable patterns into running long calculations to match for a given string.

Risks

  • Service hanging until matching is completed
  • 100% of CPU consumed
